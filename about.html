About

Datadriver was created to help non-technical individuals quickly and easily begin writing their first automated tests. It was written by Zachary Forrest and Matt Yanchek to work in conjunction with WebdriverIO and the WebdriverIO grunt plugin. Datadriver adds the following functionality:

  1. Create automated tests through a declarative API
  2. Create arbitrary viewports easily and quickly
  3. Make it easy to assert different values per browser (for example, the CSS cursor property is different in Chrome and Firefox).
  4. Write your tests with a simple step-by-step flow

We understand that this library isn't for everyone, but we've found that it helps non-technical people to brave the scary world of automated test development where manual testing is all that existed before. We didn't set out to do any thing amazing, but to do our best to solve a simple problem. If datadriver is something that helped you, we'd love to know about it.

If you would like to contribute, feel free to fork the repo, add your changes, and submit a pull request.
