Data Driver

A grunt-webdriver extension for writing data-driven Selenium tests with simple JavaScript data structures.

Arbitrary viewports

Data driver extends the functionality of grunt-webdriver. You can declare any breakpoint you need to use in your tests. 

webdriver: {
  options: {
    desiredCapabilities: {
      browserName: 'chrome',
      name: 'Documentation Tests',
      viewports: {
        small: {
          width: 430,
          height: 1000
        },
        medium: {
          width: 700,
          height: 1000
        },
        large: {
          width: 1280,
          height: 1000
        }
      }
    }
  }
}
