Data Driver

A grunt-webdriver extension for writing data-driven Selenium tests with simple JavaScript data structures.

Selenium tests without complicated code

Datadriver delivers the WebdriverIO API in easy-to-swallow data structures. Your tests follow a logical step sequence of objects, arrays, and key-value pairs.

Enable non-developers to do more in less time

Datadriver was built for the non-developer. There's no complicated logic, just simple declarative steps. Datadriver will transform those steps to actual tests executed in a browser.

Example test

This is an example of a simple test built with datadriver. Just give a quick look-over right now, we'll give you the working code in the Quick Start section. 

'use strict';


var drive = require('datadriver');


drive.go({


  "config": {
    "description": "Duck Duck Go Test",
    "suites": [
      {
        "description": "Search for DataDriver.io",
        "tests": ["duck-duck-go"],
        "viewports": { small: true }
      }
    ]
  },


  "duck-duck-go": {
    "steps": [
      {
        "selector": "#search_form_input_homepage",
        "execute": [{ "action": "type", "text": "Datadriver.io" }]
      },
      {
        "selector": "#search_button_homepage",
        "execute": [{ "action": "click" }],
      },
      {
        "verify": [
          {
            "method": "verifyPageTitle",
            "asserts": { "title": "Datadriver.io at DuckDuckGo" }
          }
        ]
      }
    ]
  }


});

Run it

grunt webdriver --url=https://duckduckgo.com
