executeAction()

Be advised that the name of this function could possibly change.

The executeAction() method doesn't do a whole lot on its own. The method takes a single object with a particular structure and runs it. For example: 

'use strict';


var drive = require('datadriver');


drive.executeAction({
  "execute": {
    "action": "redirect",
    "page": "http://google.com"
  }
});

When you run this file, the output will look something like this (this test would run against localhost since we don't pass a --url option here): 

$ grunt webdriver:actions
Running "webdriver:actions" (webdriver) task


âš™ redirecting to http://google.com.


0 passing

You probably won't need to ever call this function directly, and we would actually recommend not calling this directly unless you have a specialized use case.
