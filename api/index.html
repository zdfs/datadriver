Datadriver API Documentation

Welcome to the documentation for the datadriver API. Please note that this API is not complete. We don't have any of the Appium bindings, for example, as well as many other methods. This project is a work in progress.

You should also be aware that you should consider this software in beta. Naming conventions, how things work under the hood and API additions will all be considered par for the course until we hit 1.0. If you have an issue or a request, you can file it here.
