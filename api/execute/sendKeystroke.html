sendKeyStroke()

The sendKeyStroke() method maps directly to keys() method on WebdriverIO. This action allows you to send a sequence of keystrokes to the active element, including specialty keys like Left Arrow or Backspace. A list of the supported keys are here. 

<input type="text" id="text1" />
'use strict';


var drive = require('datadriver');


drive.execute({
  "execute": {
    "action": "redirect",
    "page": "api/execute/sendKeystroke.html"
  }
});


drive.execute({
  "selector": "#text1",
  "execute": {
    "action": "type",
    "text": "test"
  }
});


drive.execute({
  "selector": "#text1",
  "execute": {
    "action": "sendKeystroke",
    "keystrokes": "Equals"
  }
});

And here is our expected output:

$ grunt webdriver:execute --url=http://datadriver.io
Running "webdriver:execute" (webdriver) task


⚙ redirecting to "http://datadriver.io/api/execute/sendKeystroke.html"
⚙ executing "setValue" on "#text1".
⚙ sending keystroke(s) Equals
