setup()

The setup() method should never be called directly. We'll probably make this method private in the future, but we'll document what it does so you know what's going on.

First, we actually call this method for you when you require the datadriver module. What it does is expose the Chai.js assertion library with promises support. We expose the should() interface and create a custom WebdriverIO command that allows us to have an action that stores information to LocalStorage.
