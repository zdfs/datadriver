verifyState()

The verifyState() method is capable of checking to see if an element is isEnabled, isExisting, isSelected, or isVisible. We'll work our way through each state with examples and tests.

isEnabled

The isEnabled state maps directly to the isEnabled() method in WebdriverIO. It returns true or false if the selected DOM element is enabled. 

<input id="input1" type="text" placeholder="enabled" />
<input id="input2" type="text" placeholder="disabled" disabled />
<input id="input3" type="text" placeholder="disabled" disabled="disabled" />

Let's write a test to verify that these inputs are enabled or disabled.

'use strict';


var drive = require('datadriver');


drive.verify({


  "steps": [
    {
      "execute": [
        {
          "action": "redirect",
          "page": "api/verify/verifyState.html"
        }
      ]
    },
    {
      "selector": "#input1",
      "verify": [
        {
          "method": "verifyState",
          "asserts": {
            "isEnabled": true
          }
        }
      ]
    },
    {
      "selector": "#input2",
      "verify": [
        {
          "method": "verifyState",
          "asserts": {
            "isEnabled": false
          }
        }
      ]
    },
    {
      "selector": "#input3",
      "verify": [
        {
          "method": "verifyState",
          "asserts": {
            "isEnabled": false
          }
        }
      ]
    }
  ]


});

And here is our test output:

$ grunt datadriver:verify --url=http://datadriver.io
Running "datadriver:verify" (datadriver) task


⚙ redirecting to http://datadriver.io/api/verify/verifyState.html
✓ the "#input1" element is enabled on the page
✓ the "#input2" element is not enabled on the page
✓ the "#input3" element is not enabled on the page

isSelected

The isSelected state maps directly to the isSelected() method in WebdriverIO. It returns true or false if an option element or input of type checkbox or radio is currently selected. 

<select id="select1">
  <option value="Andrew Hudgins">Andrew Hudgins</option>
  <option value="Sharon Olds" selected>Sharon Olds</option>
  <option value="Yusef Komunyakaa">Yusef Komunyakaa</option>
</select>

Let's write a test to verify that the right option is selected.

'use strict';


var drive = require('datadriver');


drive.verify({


  "steps": [
    {
      "execute": [
        {
          "action": "redirect",
          "page": "api/verify/verifyState.html"
        }
      ]
    },
    {
      "selector": "#select1 option:nth-of-type(1)",
      "verify": [
        {
          "method": "verifyState",
          "asserts": {
            "isSelected": false
          }
        }
      ]
    },
    {
      "selector": "#select1 option:nth-of-type(2)",
      "verify": [
        {
          "method": "verifyState",
          "asserts": {
            "isSelected": true
          }
        }
      ]
    },
    {
      "selector": "#select1 option:nth-of-type(3)",
      "verify": [
        {
          "method": "verifyState",
          "asserts": {
            "isSelected": false
          }
        }
      ]
    }
  ]


});

And here is our test output:

$ grunt datadriver:verify --url=http://datadriver.io
Running "datadriver:verify" (datadriver) task


⚙ redirecting to http://datadriver.io/api/verify/verifyState.html

Call verifyState() directly

You can also call verifyState() directly in your tests for even more granularity. The method takes an object with four keys. 

'use strict';


var drive = require('datadriver');


drive.verifyState({
  selector: '#zeroOpacity',
  assertKey: 'isVisible',
  assert: false,
  mode: [ 'equal' ]
});
